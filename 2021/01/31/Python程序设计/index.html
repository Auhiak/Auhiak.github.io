    • 第一章 Python基本语法元素

    -1.1 程序设计基本方法

    -计算机与程序设计

    1. 计算机是指根据指令操作数据的设备

    2. 计算机的概念

    功能性

    对数据的操作，表现为数据计算、输入输出处理和结果存储等

    可编程性

    根据一系列指令自动地、可预测地、准确地完成操作者的意图

    -编译和解释

    -源代码

    采用某种编程语言编写的计算机程序，人类可读

    -目标代码

    计算机可直接执行，人类不可读(专家除外)

    编译

    将源代码一次性转换成目标代码的过程，一次性翻译，之后不再需要源代码(类似英文翻译)

    解释

    将源代码逐条转换成目标代码同时逐条运行的过程,每次程序运行时随翻译执行(类似实时的同声传译)

    根据执行方式不同，编程语言分为两类

    -静态语言

    使用编译执行的编程语言

    -脚本语言

    使用解释执行的编程语言

    -程序的基本编写方法

    -I Input 输入，程序的输入

    -P Process 处理，程序的主要逻辑

    -O Output输出，程序的输出

    编程解决问题的步骤
    6个步骤

    -分析问题：分析问题的计算部分，想清楚

    -划分边界：划分问题的功能边界，规划IPO

    -设计算法：设计问题的求解算法，关注算法

    -编写程序：编写问题的计算程序，编程序

    -调试测试：调试程序使正确运行，运行调试

    -升级维护：适应问题的升级维护，更新完善
